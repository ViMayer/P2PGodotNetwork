# Godot Peer-to-Peer Multiplayer Network

This is a network structure developed in Godot Engine 4.5.1 that allows for P2P hosting and connecting

## Key Features

* **Lobby Keys:** Allows for easy server joining using lobby keys
* **User Information:** Saves the current and connected users information in the Network global script, making it easier to manage multiplayer scenes

## How to Run the Project

1. **Clone or Download:** Obtain the repository by cloning it via Git or downloading the ZIP file.
2. **Open in Godot Engine:** Load the project in your [Godot Engine](https://godotengine.org) installation.
3. **Execute:** Press <kbd>F5</kbd> or click `Run Project` in the Godot editor.

## Controls

* <kbd>Middle Mouse Button</kbd> or <kbd>R3</kbd> for debugging.
* <kbd>Esc</kbd> or <kbd>Options</kbd> for toggling the lobby menu.

## Testing
To test the multiplayer functionality in your local machine:
Go to `Debug` > `Customize Run Instances`, then enable `Enable Multiple Instances` and set the number of instances you want to run simultaneously.
